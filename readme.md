# LCD Konsole – Stationeers Mod

## Eigenschaften
- Geneigte Floor-Konsole
- LCD-Logik (IC10 / Network Write)
- Ein Port (Strom = Daten)
- Keine Karten
- Keine Karten-Slots
- Feste weiße Titelzeile (Objektname)
- 8 Zeilen gesamt, 7 nutzbar
- Stackgröße: 5
- Electronics Printer
- Electronics Research

## Zeilenlayout
[ Titel ]
Label 1
Wert 1
Label 2
Wert 2
Label 3
Wert 3
Reserve

## Setting-Regeln
Setting 0: Titel (gesperrt)
Setting 1–7: frei nutzbar

## Zeichenlimit
Empfohlen max. 32 Zeichen pro Zeile.
Kein Umbruch, kein Scrollen.

## IC-Grundsatz
IC rechnet – Konsole zeigt.
Text und Zahlen getrennt senden.
